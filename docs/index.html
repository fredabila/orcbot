Get started GitHub

Getting started

  1. Install dependencies: npm install
  2. Build: npm run build
  3. Run the agent: orcbot run
  4. Open the TUI: orcbot ui

Configuration

Configuration is resolved in this priority order:

  • Environment variables
  • ./orcbot.config.yaml
  • ~/orcbot.config.yaml
  • ~/.orcbot/orcbot.config.yaml

Autonomy & heartbeat

Heartbeat triggers proactive work when idle, with backlog limits, stall detection, and recovery.

Skills & plugins

Core skills live in src/core/Agent.ts. Hot-load plugins from ~/.orcbot/plugins.

Infrastructure map

Channels → Agent Core → Skills & Tools → Providers

Channels

Telegram and WhatsApp are supported out of the box. Add more in src/channels.

Search reliability

Configure Serper, Brave Search, or SearxNG for resilient web search.

Troubleshooting

Check logs in logs/ and the memory store in ~/.orcbot for diagnostics.
