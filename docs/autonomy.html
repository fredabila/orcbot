Autonomy

Heartbeat-driven autonomy.

The scheduler keeps OrcBot active even when you are away.

Heartbeat loop

Heartbeat runs on the scheduler tick and enqueues proactive tasks when idle.

Backlog limit

Use autonomyBacklogLimit to cap queued proactive tasks.

Stall detection

Long-running actions are failed via maxActionRunMinutes.

Stale recovery

Stale in-progress actions are marked failed after maxStaleActionMinutes.

Autonomy checklist

  • Enable autonomyEnabled
  • Set a heartbeat interval that fits your workload
  • Monitor logs for proactive task output
