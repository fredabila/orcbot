Configuration

Control how OrcBot behaves. Configuration is file-backed, environment-aware, and editable from the TUI.

üìÅ Resolution Order

OrcBot looks for configuration in this order (first found wins):

  1. Environment variables
  2. ./orcbot.config.yaml (current directory)
  3. ~/orcbot.config.yaml (home directory)
  4. ~/.orcbot/orcbot.config.yaml (data directory)

üêº Lightpanda Browser

Lightpanda is a lightweight headless browser that uses 9x less RAM and is 11x faster than Chrome. Perfect for low-memory VPS deployments.

Quick Setup

# Install Lightpanda
orcbot lightpanda install

# Start the browser server (background)
orcbot lightpanda start -b

# Enable as default browser engine
orcbot lightpanda enable

# Check status
orcbot lightpanda status

CLI Commands

  • orcbot lightpanda install ‚Äî Download and install Lightpanda
  • orcbot lightpanda start [-b] ‚Äî Start browser server (-b for background)
  • orcbot lightpanda stop ‚Äî Stop background server
  • orcbot lightpanda status ‚Äî Check installation and server status
  • orcbot lightpanda enable ‚Äî Set Lightpanda as default browser
  • orcbot lightpanda disable ‚Äî Switch back to Playwright/Chrome

Config Options

  • browserEngine ‚Äî playwright (default) or lightpanda
  • lightpandaEndpoint ‚Äî CDP endpoint (default: ws://127.0.0.1:9222)
  • lightpandaPath ‚Äî Path to binary (auto-set by installer)
