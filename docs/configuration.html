Configuration

Control how OrcBot behaves.

Configuration is file-backed, environment-aware, and editable from the TUI.

Resolution order

  1. Environment variables
  2. ./orcbot.config.yaml
  3. ~/orcbot.config.yaml
  4. ~/.orcbot/orcbot.config.yaml

Key settings

Core

  • agentName
  • modelName
  • maxStepsPerAction
  • maxMessagesPerAction
  • messageDedupWindow

Autonomy

  • autonomyEnabled
  • autonomyInterval
  • autonomyBacklogLimit
  • maxActionRunMinutes
  • maxStaleActionMinutes

Providers

  • llmProvider
  • openaiApiKey
  • googleApiKey
  • bedrockRegion
  • bedrockAccessKeyId
  • openrouterApiKey
  • openrouterBaseUrl

Search

  • serperApiKey
  • braveSearchApiKey
  • searxngUrl
  • searchProviderOrder

Channels

  • telegramToken
  • whatsappEnabled
  • whatsappAutoReplyEnabled
  • whatsappStatusReplyEnabled

Decision Pipeline

  • skillRoutingRules
  • autopilotNoQuestions
  • autopilotNoQuestionsInterval

TUI management

Open the TUI with orcbot ui to configure models, tooling, and connections without editing files.
