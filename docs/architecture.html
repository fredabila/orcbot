Architecture

Understand the system layout.

Core orchestration, memory, tools, and external providers.

Core flow

Channels → Agent Core → Skills & Tools → Providers
Telegram / WhatsApp / CLI • LLMs • Search providers • Storage

Agent Core

DecisionEngine + SimulationEngine orchestrate every task with plans and guardrails.

Memory & Queue

MemoryManager stores short/episodic context. ActionQueue persists tasks.

Scheduler

Heartbeat triggers autonomy tasks and monitors for stalls.

Skills

SkillsManager loads core tools and hot-reload plugins at runtime.

Web tools

WebBrowser handles navigation, snapshots, and search providers.

Providers

MultiLLM routes to OpenAI, Gemini, or Bedrock with fallback logic.

Data paths

  • ~/.orcbot: memory, profiles, plugins, downloads
  • logs/: runtime logs
  • orcbot.config.yaml: configuration source
