Quick Start

Get OrcBot running locally in under 5 minutes.

ğŸ“‹ Prerequisites

  • Node.js 18+ (we recommend 20 LTS)
  • Git
  • OpenAI or Google Gemini API key

Step 1: Clone & Install

git clone https://github.com/fredabila/orcbot.git
cd orcbot
npm install
npm run build

Step 2: Run Setup Wizard

The setup wizard will guide you through configuring API keys and preferences.

npm run dev setup
# Or if installed globally:
orcbot setup

Step 3: Start OrcBot

# Run the agent loop
npm run dev run

# Or use the interactive TUI
npm run dev ui

# Or install globally and run anywhere
npm install -g .
orcbot run
