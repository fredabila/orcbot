Getting started

Launch OrcBot in minutes.

Install dependencies, run the agent, and open the TUI dashboard.

Install & build

  1. Install dependencies: npm install
  2. Build the CLI: npm run build
  3. Run the agent: orcbot run

Run the TUI

Use orcbot ui to configure models, connections, and tooling.

Push a task

Trigger work immediately with orcbot push "Your task" -p 10.

Verify logs

Check logs/combined.log for runtime traces and diagnostics.

Quick checklist

  • Set API keys via orcbot ui or config files.
  • Enable channels you need (Telegram / WhatsApp).
  • Confirm heartbeat settings for autonomy.
