Skills & Plugins

Extend OrcBot with new tools.

Core skills live in the agent; plugins are hot-loaded without restarts.

Core skills

Defined in src/core/Agent.ts. Core skills include web search, browser automation, and channel messaging.

Plugin directory

Store plugins in ~/.orcbot/plugins or ./plugins.

Self repair

Broken plugins trigger self_repair_skill and reload automatically.

Skill builder

Use the TUI to build new skills from specifications.

Plugin contract

module.exports = {
  name: 'skill_name',
  description: 'What it does',
  usage: 'skill_name(arg)',
  handler: async (args, context) => {
    return 'result';
  }
};
