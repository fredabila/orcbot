Operations

Run OrcBot in production.

Logs, diagnostics, and operational checklists for reliable uptime.

Logs

Review logs/combined.log for runtime events, errors, and search telemetry.

Data locations

State is stored under ~/.orcbot (memory, profiles, plugins, downloads).

Backups

Back up memory.json, actions.json, and USER.md for continuity.

Search failures

Configure Serper/Brave/SearxNG for reliable search and avoid loop retries.

Troubleshooting checklist

  • Confirm API keys are set in config or env.
  • Restart the agent after changing channel settings.
  • Inspect failed actions in actions.json.
