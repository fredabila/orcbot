What the canvas is for

Purpose
The canvas is the live operator workspace. It keeps queue state, live events, chat, services, and the payload inspector on one screen so you can monitor or steer OrcBot without switching between tabs.
Use it when
You need to watch the agent react in real time, inspect a task or service payload, or let an external A2UI surface push tasks and events into the dashboard bridge.
